# 設計書：LINE Sticker Splitter

v2: AI(rembg)除去廃止、カラーベース高品質化

## 概要

| 項目 | 内容 |
|------|------|
| 目的 | 一覧画像→分割→背景除去→LINE仕様→ZIP |
| 技術 | Python / Flask / Pillow / OpenCV / NumPy / Vanilla JS |

## rembg廃止の理由

rembgは被写体セグメンテーションモデルのため
テキスト（了解！最高！等）やキラキラ・ハートを
背景扱いして消してしまう。スタンプとして致命的。

## 背景除去アルゴリズム (v2)

    [1] コーナー4箇所x8pxサンプリング → 背景色中央値推定
    [2] HSV空間でマスク生成 + RGB距離フォールバック
    [3] モルフォロジー Close(ゴミ除去) + Open(穴埋め)
    [4] ガウシアンブラーでアルファ境界をソフトに

文字・装飾は背景色と異なるため全て保持される。
